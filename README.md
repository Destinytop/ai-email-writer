# AI Email Writer

Write professional emails in seconds with AI-powered assistance.

## Features

- ✨ Smart email composition
- 🎯 Tone adjustment
- ✅ Grammar & spell check
- 📧 Multiple templates
- 🔒 Privacy focused

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

```bash
vercel --prod
```
